#Screenshots

## Installation

```
git clone git@github.com:thisislawatts/Screenshots.git
npm install
```

You'll then need to create a `browserstack.json` file

```
{
  "username": "foo",
  "password": "bar"
}
```


## Usage

```
./browsersnap.js --version
./browsersnap.js browsers
./browsersnap.js get [url]
```
